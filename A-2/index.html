JavaScript Decision Making Questions

Q1. Find Grades

Your school has the following grading system based upon the marks (M) obtained by a student:

  • If M ≤ 10, the grade will be E.
  • If 11 ≥ M ≤ 20, the grade will be D.
  • If 21 ≥ M ≤ 30, the grade will be C.
  • If 31 ≥ M ≤ 40, the grade will be B.
  • If 41 ≥ M ≤ 50, the grade will be A.

Your friend will enter his marks out of 50, and your task is to print his grades using a switch statement.

Input Format: An integer M (0 ≤ M ≤ 50)

Output Format: Return the grade obtained by your friend.

Sample Input: 1

Sample Output: E

Q2. Get Value

You are provided with a table containing some characters and their corresponding values. Your task will be to find the value corresponding to an input character.

  • P or p - PrepBytes
  • Z or z - Zenith
  • E or e - Expert Coder
  • D or d - Data Structure

Input Format: A character C (C ∈ [P, p, Z, z, E, e, D, d])

Output Format: Return the value corresponding to the input character.

Sample Input: E

Sample Output: Expert Coder

Q3. Find the maximum out of three numbers

Take three numbers and print the largest number among them. If all three are the same, print -1.

Input Format: Three integers A, B, and C (1 ≤ A, B, C ≤ 10000)

Output Format: Return the maximum number, or -1 if all are equal.

Sample Input: 2 5 4

Sample Output: 5

Q4. Second Smallest

You are given 3 distinct integers X, Y, and Z. Find and return the second smallest integer.

Input Format: Three space-separated integers X, Y, and Z (1 ≤ X, Y, Z ≤ 500; all are distinct)

Output Format: Return the second smallest integer.

Sample Input: 2 9 23

Sample Output: 9

Q5. Check whether the triangle is Acute or Obtuse

Write a program that takes three angles and checks whether the triangle is acute or obtuse.

Input Format: Three integers A, B, and C (0 ≤ A, B, C ≤ 180)

Output Format: Return "acute" or "obtuse".

Sample Input: 60 100 20

Sample Output: obtuse
